Privacy and Information

Keep reading for information on the privacy policy, page information, API information, and more.

Privacy Policy

We use Google Analytics for tracking the use of the pages across this whole site.

The collected information includes:

  • Location information.
  • Time spent on each page.
  • Time before leaving a page.
  • Interaction such as scrolls, clicks etc

This information is for our own personal use and will not be shared with anyone else by us.

We use this information to allow us to continually improve our site.

If you have any concerns, feel free to contact us here.

API Information

This website uses data from the Bored API.

This information is used to fill the cards which provides activity suggestions.

Other Information
